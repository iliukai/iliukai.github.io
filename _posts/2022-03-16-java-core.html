入门3点

  • class后面的名字是类名
  • 类名必须与源文件的文件名相同，文件名后缀必须是小写的java
  • main方法是Java程序执行的入口

Hello World

// print 'hello world'
public class HelloWorld {
    public static void main(String[] args){
        System.out.println("Hello World !");
    }
}
